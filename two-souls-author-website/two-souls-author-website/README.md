# The Two Souls Series Author Website

A cinematic luxury Next.js website for Shadow Wolf.

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Deploy to Vercel
1. Upload this folder to GitHub, or drag the folder into Vercel.
2. Framework preset: Next.js
3. Build command: `npm run build`
4. Output: default
5. Deploy.

## Images
Book covers are already placed in:
`public/books/book-1.png` through `book-6.png` and `complete-series.png`.
Replace them with the same filenames if you update the covers.
